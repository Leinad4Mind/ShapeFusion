ShapeFusion
===========

Serve para combinar objectos. Para quando temos mais que um objecto e queremos combinar vários num só.


Por fazer
-----------------
* Creio que já está tudo feito, mas estarei disposto a melhorar se houver dicas para tal.


Como instalar
--------------

Load Automático

1. Transferir shapefusion.lua
2. Colocar esse ficheiro na pasta _autoload_ situada dentro da pasta _automation_ que por sua vez está presenta na sua pasta de instalação do aegisub


Load Manual

1. Transferir shapefusion.lua e guarde-o onde bem desejar
2. Com o Aegisub aberto, quando desejar usá-lo terá de clicar em Automatização -> Automatização...
3. Clicar de seguida em _Adicionar_ e ir ao local onde se encontra o shapefusion.lua


Como usar
---------

Basta seleccionar as linhas dos objectos que se pretende combinar, executar a macro e será então criada uma nova linha com os objectos seleccionados.
Caso para dizer: FUSÃO! :P
